# TruEstate - Backend

This is the backend for the TruEstate Retail Sales Management System assignment.

## Overview

- Node.js + Express backend that serves `/api/sales`
- Supports full-text search, multi-select filters, range filters, sorting, pagination
- Loads dataset from `backend/data/sales_dataset.json` (generated from CSV)
